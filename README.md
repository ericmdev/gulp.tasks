Gulp Tasks
==========

Gulp tasks in CoffeeScript - configured in YAML.

Usage
-----

Clone repo into a new directory in your project:
    
    $ git clone [[repo]] ./gulp
    $ cd ./gulp

Edit `gulpfile.js` and `config.yml` as required:

Now run gulp - the `default` task is `develop`:

    $ gulp [[task name]]

Build:
    
    $ gulp build

Test
----

Test tasks using npm:

    $ npm test
